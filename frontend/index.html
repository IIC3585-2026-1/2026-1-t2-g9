🧠 Motor de consultas

Este proyecto implementa un motor de consultas en memoria inspirado en SQL, utilizando programación funcional. Las consultas se construyen como un pipeline de transformaciones.

  • query(data): punto de entrada
  • where: filtra datos
  • select: selecciona campos
  • orderBy: ordena resultados
  • groupBy: agrupa datos
  • aggregate: aplica funciones a grupos
  • execute: ejecuta la consulta

Internamente, cada operación agrega una función a un pipeline. La ejecución ocurre solo al llamar a execute(). 

query(data)
  .where(m => m.rating > 8)
  .select(['title', 'rating'])
  .orderBy('rating', 'desc')
  .execute()

Esto se traduce en la aplicación secuencial de funciones usando reduce.
